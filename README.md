# ⚽ Predictor de Resultados - Mundial 2026

App web para predecir resultados del FIFA World Cup 2026 usando XGBoost y Random Forest.

## 🚀 Características

- Predicciones de resultados (1/X/2)
- Estimación de goles esperados
- Dashboard interactivo
- Tabla de predicciones filtrable
- Histórico de resultados

## 📊 Tecnología

- **Model:** XGBoost + Random Forest
- **Features:** ELO ratings, últimos 5/10 partidos, estadísticas
- **App:** Streamlit
- **Hosting:** Streamlit Cloud
